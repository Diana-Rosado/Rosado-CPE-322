Below are commands that are ran in the vs code terminal.

```
$ hostname  //displays hostname
$ env   //displays environment variables
$ ps    //displays info about processes running on system
$ pwd   //displays curernt working directory
```

```
$ git clone https://github.com/kevinwlu/iot.git //clones github repo
$ cd iot    //goes to folder iot
$ ls        //displays content within iot folder
$ cd        //changes current directory to home directory
$ df        //displays info about disk space usage
```
<img src="https://github.com/Diana-Rosado/Rosado-CPE-322/blob/main/Labs/Lab2/clone2df.png" width="800" height="600">

```
$ mkdir demo //creates a folder called demo
$ cd demo   //goes to folder
$ nano file //opens nano text editor and creates a file named 'file'
$ cat file  //displays contents of file
$ cp file file1 //creates a copy of the file and names it file1
$ mv file file2 //renames original file -> file2
$ rm file2  //deletes file2
```
